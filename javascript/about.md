Javascript基礎
=============

機能
----

* HTML操作
* クライアントサイドの実行
* インタープリタ
* 型は三種。コンテキストで自動的に決まる
    * 数値型
    * 論理型
    * 文字列
* オブジェクトベース(≠オブジェクト指向)。クラスは予め用意されたものを利用

制約
----

* 一度定義されたプロパティは変更出来ない
* 予約語がある



オブジェクト
-----------

* オブジェクトの成り立ち
    * プロパティ(属性)
    * 関数(メソッド)
    * 変数(メンバ変数)
* 階層
    * window
        * document
            * anchor
            * form
                * button
                * checkbox
                * ....
            * link
            * location
       * frame
       * history
